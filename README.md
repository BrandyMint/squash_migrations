# SquashMigrations

Squash migrations to last migration.

## Installation

Add this line to your application's Gemfile:

    gem 'squash_migrations'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install squash_migrations

## Usage

    rake db:squash_migrations
